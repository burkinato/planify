# ISO 7010 Safety Signs Skill

## Overview

ISO 7010:2019 standardizes safety signs for accident prevention, fire protection, health hazard information, and emergency evacuation.

---

## Sign Design Rules

1. **Proportions:** Symbol should occupy min 50% of visible area
2. **Contrast:** Min 60% contrast ratio
3. **Mounting height:** 1400-1600mm (center) for standing viewing
4. **Illumination:** Min 100 lux for indoor, 50 lux for outdoor
5. **Viewing distance:** Symbol height (mm) / 125 = max viewing distance (m)

---

## Integration Guidelines

When adding ISO 7010 signs:

- Use official symbol codes (W001, P001, etc.)
- Maintain proportions per standard
- Consider contrast and lighting
- Place at appropriate height
- Use consistent color values
- Document usage location in system
